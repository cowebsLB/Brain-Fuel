# Feature Breakdown

## Brand and Hero

- Sticky header with logo and brand title.
- Hero section with tagline and CTA actions.
- Optional steam accent animation for brand atmosphere.

## Menu Categories

- Reusable category cards for:
  - Hotties
  - Coldies
  - Desserts
  - Tea & Water
- Item rows with consistent typography and price formatting.

## Parallax Separators

- One separator block after each category card.
- Local section-specific imagery with dark overlay and label.

## Navigation

- Sticky bottom mobile category nav.
- Active tab indicator based on visible section using intersection observers.

## Motion and Accessibility

- Page reveal stagger.
- Card hover/focus and CTA interactions.
- Reduced-motion override that disables non-essential motion/parallax.
