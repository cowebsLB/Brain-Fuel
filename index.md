# Brain Fuel

A React + Vite single-page digital menu for Brain Fuel cafe, inspired by the local Stitch mockups and adapted into a maintainable production-ready frontend structure.

## Features

- Chalkboard cafe visual system with reusable design tokens.
- Responsive menu page with semantic sections and sticky mobile category nav.
- Structured menu data rendering (single source of truth).
- Subtle motion system: reveal, hover/focus, CTA micro-interactions, active tab indicator.
- Parallax separators between sections using local stitched food photos.
- Reduced-motion accessibility fallback.

## Setup

```bash
npm install
```

## Usage

```bash
npm run dev
npm run build
npm run preview
```

## Tech Stack

- Node.js + npm
- React
- Vite
- CSS (tokenized, modular files)
